# Embeddable web apps
A collection of web apps that you can embed in your web pages.
Intended for embedding in wiki pages for online Learning Management Systems such as Canvas.

## Python IDE
Embed a minimal Python IDE with code editor and output in your page.

### Features
* Allows starter code in URL
* Editing and running Python scripts
* Save and load .py files
* Change font size
* Reset button to return to starter code

### Instructions
1. Access https://jcjeffreys.github.io/embeddable-web-apps/python-ide.html
2. Add your starter code
3. Use the buttons to copy the embed frame or URL
